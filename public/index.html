Hello! Thanks for checking out Bird-ID, a website/Discord bot for aspiring ornithologists.

You probably want to join our Discord server, add this bot to an existing server, or practice on the web.

Usage

This bot uses bird images and sounds from Cornell's Macaulay Library to help you practice bird identification!

Discord Prefixes (case insensitive): b!, b., b#, o>

Quickstart:

  • Use b!bird or b!b for a new bird image.
  • Use b!check [answer] or b!c [answer] to check your answer.
  • Use b!hint or b!h to get a hint.
  • Use b!skip or b!sk to skip the current bird.

See the full help text for info on available commands.

Science Olympiad

Science Olympiad is an academic competition for K-12 students where they compete in a variety of science and engineering topics.

This bot was originally created for the Science Olympiad's Ornitholigy event and its National Bird List, but can be adapted with custom bird lists for other purposes. This bot is not affiliated with Science Olympiad. Visit SOINC's official website for more information about Science Olympiad and the Ornithology event.

Full Help Text:

  • Birds

    • bird - Sends a random bird images for you to ID
    • goatsucker - Sends a random goatsucker to ID
    • song - Sends a random bird song for you to ID

  • Check:

    • check - Checks your answer

  • Hint:

    • hint - Gives first letter of current bird

  • Meta:

    • botinfo - Gives info on bot, support server invite, stats
    • ignore - Ignore all commands in a channel
    • invite - Get the invite link for this bot
    • leave - Remove the bot from the guild
    • ping - Pings the bot and displays latency

  • Other:

    • filters - Lists available Macaulay Library filters.
    • info - Gives an image and call of a bird
    • list - DMs the user with the appropriate bird list.
    • meme - Sends a funny bird video!
    • taxon - DMs the user with the appropriate bird list.
    • wikipedia - Fetch the wikipedia page for any given argument

  • Race:

    • race - Base race command

  • Score:

    • leaderboard - Top scores
    • missed - Top incorrect birds
    • score - Total correct answers in a channel or server
    • streak - Gives your current/max streak
    • userscore - How many correct answers given by a user

  • Sessions:

    • session - Base session command

  • Skip:

    • skip - Skip the current bird to get a new one

  • States:

    • custom - Sets your custom bird list
    • set - Sets your state

  • Stats:

    • export - Exports bot data as a csv
    • frequency - Gives info on command/bird frequencies
    • stats - Gives statistics on different topics

  • Voice:

    • back - Skip back 5 seconds
    • disconnect - Disconnect from voice
    • forward - Skip forward 5 seconds
    • pause - Pause playing
    • play - Play a sound
    • replay - Replay
    • stop - Stop playing

  • No Category:

    • help - Shows this message

Type b!help command for more info on a command. You can also type b!help [category] for more info on a category.
